Everything you need for autonomous development

A complete pipeline from idea to merged PR, no human in the loop required.

Multi-Agent Pipeline

Specialized agents for each phase: analyst, architect, designer, planner, developer, reviewer.

Per-Task Review Loop

Every developer task is reviewed automatically. Revisions run until PASS.

Pluggable Backends

Azure Blob Storage + Queues or GitHub Issues + branches. Switch with one env var.

Zero Babysitting

Serial dispatch, automatic dead-letter handling, and fan-out planning. Fully autonomous.

Claude-Native

Built on Claude Code CLI. Each agent uses the best-fit Claude model.

Open Source

MIT licensed, self-hostable, transparent. Fork it, extend it, own it.

From idea to code in three steps

No configuration, no babysitting. Just describe what you want.

01

Brainstorm

Describe your feature in a conversation. AgentHarness writes the brief and uploads it to storage.

02

Agents Work

The pipeline runs analyst → architect → planner → developer → reviewer autonomously, end to end.

03

Code Ships

Implementation lands in your repo on a feature branch, reviewed and ready to merge.

Up and running in minutes

Install, initialise, and start your first autonomous pipeline with three commands.

pip install agentharness
agentharness init
