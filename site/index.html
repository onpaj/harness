How It Works

Describe the feature once. The pipeline handles the rest.

  1. 💡
    Brainstorm Explore and define what to build
  2. 📋
    Planner Break the feature into a detailed spec
  3. 🏗️
    Architect Design the structure and integration points
  4. 🎨
    Designer Define UI components and interactions
  5. ⚡
    Developer(s) Implement code with tests in parallel
  6. 🔍
    Reviewer Verify quality, security, and correctness
  7. ✅
    Done Reviewed implementation ready to merge

Why AgentHarness

  • Save hours per feature

    Describe what you want once. The pipeline delivers a reviewed implementation without further input.

  • No guiding required

    Agents make architectural and design decisions autonomously. You review the result, not the process.

  • No correcting mistakes

    Each stage's output is validated by the next. Errors are caught before they reach you.

  • No context-switching

    Queue a feature and return to your current work. The pipeline runs without your attention.

Quick Start

Prerequisites: Claude Code with AgentHarness skills installed.

  1. Ensure Claude Code is running with AgentHarness skills installed
  2. In Claude Code, describe the feature you want built
  3. Run the brainstorm skill to explore and define it: /brainstorm
  4. When ready, kick off the full pipeline: /implement
